## FES - Front-End Stage

## Features:
- Autoreload
- [Stylus](http://stylus-lang.com/) for styling
- [Pug](https://pugjs.org) for templates

### Start:
1. `bower install`
2. `npm install`
3. `gulp`

Project will run on `localhost:4000`

### Dependencies:
- node 8.8.1 (tested on)
- bower
  - `npm install -g bower`
- gulp
  - `npm install -g gulp-cli`

#### Dev Dependencies:
- Those above
