## FES - Front-End Stage

### Start:
1. `bower install`
2. `npm install`
3. `gulp`

### Dependencies:
- Ruby
- Sass 3.4.20+
  - `gem install sass`
- node 5.4.0
- npm 3.5.2
- bower
  - `npm install -g bower`
- gulp
  - `npm install -g gulp`

#### Dev Dependencies:
- Those above
- Bourbon 4.2.6+
  - `gem install bourbon`
- Neat
  - `gem install neat`
