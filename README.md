## FES - Front-End Stage

## Features:
- Autoreload
- [Stylus](http://stylus-lang.com/) for styling
- [Pug](https://pugjs.org) for templates

### Start with livereloading:
1. `npm install`
2. `gulp startdev`

Project will run on `localhost:5000`

### Just generate build:
1. `npm install`
2. `gulp`

### Dependencies:
- node 11.15.0 (tested on)
- gulp
  - `npm install -g gulp-cli`
